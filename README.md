# BOM Manager - Walmart Sourcing

A Bill of Materials (BOM) management tool for Walmart Sourcing teams. Built with vanilla HTML, JavaScript, and Tailwind CSS.

## Features

- Create and manage BOMs with parts, packaging, and overhead cost sections
- Department and category selection
- Export to Excel
- Email integration
